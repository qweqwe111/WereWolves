接口返回参数: Object

属性名

类型

描述

success

string

请求是否成功

message

string

返回的信息(作为请求失败后所提示的信息)

data

Object

请求成功后所返回的数据

请求例子:

    $.ajax('/api/Do/CreateRoom', {

        dataType: 'json',

        type: 'post',

        data: { roles: '1,1,1,1,2,2,2,2,3,4,5,6', Userid: 2, Pnum: 12, Victory:1},

        success: function (Backdata) {

            if (Backdata.state == 'success') {

                       //请求成功后所做的事情

            }

            else {

                Backdata.message;

                      //请求失败所提示的信息(如房间号错误等)

            }

        }

    });
